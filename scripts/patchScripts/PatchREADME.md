This folder includes scripts for patching pipelines. Execute via terminal under this directory. 

Patching is intended to upgrade OSIS pipelines' DataPrepper versions without changing their configurations. 

For a comprehensive guide, follow the 'Patching' section in the [Patching Pipelines and Container SOP](https://quip-amazon.com/vQa3AbFGRIcZ/Patching-Pipelines-and-Container-Security-SOP#temp:C:VVYca383d478f7c4fcdb038d50e2)
